# sample_homepage

### 環境構築
---
- 注意点  
staticとserverは並列な場所に配置する事  
  - 例  
  workspace/static  
  workspace/server
- 下記はクローンした時だけ実行  
```npm install```
  - windowsの場合は下記2つも  
```npm install -g webpack```  
```npm install -g webpack-cli```  

- static-build  
```npm run build```  
- server-start  
```npm run start```
